## 🛠️ 常见问题排查  
### 基础检测  
✅ 必须项检查：    
- 订阅有效期 ≥ 当前日期    
- 未使用「直连/Direct」模式    
- 节点状态显示为「正常」

### 进阶处理  
🔄 故障排除步骤：    
1. 更新订阅（客户端内点击「同步」按钮）    
2. 切换网络（4G/Wi-Fi交替测试）    
3. 重启设备 & 路由器    
4. 重装客户端（建议先备份配置）
