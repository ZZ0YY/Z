# 批量文件重命名

**API：** `POST 域名 + /api/v1/file/rename`

**说明：** 批量重命名文件，最多支持同时30个文件重命名

### Header 参数

| 名称          | 类型   | 是否必填 | 说明               |
|---------------|--------|----------|--------------------|
| Authorization | string | 必填     | 鉴权access\_token   |
| Platform      | string | 必填     | 固定为:open\_platform |

### Body 参数

| 名称       | 类型  | 是否必填 | 说明                                  |
|------------|-------|----------|---------------------------------------|
| renameList | array | 必填     | 数组,每个成员的格式为 文件ID\|新的文件名 |

### 示例

#### 请求示例
