# 移动

**API：** `POST 域名 + /api/v1/file/move`

**说明：** 批量移动文件，单级最多支持100个

### Header 参数

| 名称          | 类型   | 是否必填 | 说明               |
|---------------|--------|----------|--------------------|
| Authorization | string | 必填     | 鉴权access\_token   |
| Platform      | string | 必填     | 固定为:open\_platform |

### Body 参数

| 名称           | 类型   | 是否必填 | 说明                                           |
|----------------|--------|----------|------------------------------------------------|
| fileIDs        | array  | 必填     | 文件id数组                                     |
| toParentFileID | number | 必填     | 要移动到的目标文件夹id，移动到根目录时填写 0 |
